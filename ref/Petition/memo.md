# Meritx Pivot

upgrade existing intake tool to attorney-ready petition package system

AI-native workflow system that assembles attorney-ready petition packages under lawyer-specific playbooks.

MeritX helps immigration law firms turn client intake and evidence into attorney-ready petition packages, including:
- structured case summaries
- eligibility analysis
- form-field mapping
- exhibit indexes
- petition letter drafts
- missing evidence reports
- final attorney review workflows

2. Evidence Mapping
3. Missing Document Checklist
